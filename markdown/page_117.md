# Page 117 - Condizione di Arresto Spontaneo

da cui

$$\frac{1 - \eta}{1 - \eta_R} = \frac{f}{\rho}$$

in generale $\rho < 1$ perché il sistema è meno carico, in fatti Mirko non sta più tirando la massa $\Rightarrow R' < R$

## Quindi:

$$f - \eta \rho = \eta - \eta_R \quad \sim\!\!> \quad \eta - \eta_R = \eta + \eta \rho - \rho$$

ora divido per $\eta$:

$$\sim\!\!> \quad \eta_R = 1 + \rho - \frac{\rho}{\eta} < 0 \quad \text{condizione di arresto spontaneo}$$

Per cui

$$\frac{f}{\eta} > 1 + \rho \quad \Rightarrow \quad \frac{\eta}{\rho} < \frac{1}{1+\rho} \quad \sim\!\!> \quad \boxed{\eta < \frac{\rho}{1+\rho}} \quad \text{CONDIZIONE DI ARRESTO SPONTANEO}$$

ed essendo dato che generalmente $\rho < 1 \Rightarrow \frac{\rho}{1+\rho} < 0{,}5$ e quindi si può mai avere anche

$$\boxed{\eta < 0{,}5} \quad \text{CONDIZIONE DI ARRESTO SPONTANEO}$$
